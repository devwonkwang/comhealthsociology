# 마음의 구조와 자연어 처리

이 프로젝트는 공개 강의자료 프로젝트입니다. 향후 공개 Ebook으로 발전시켜 나가려 합니다.  
자연어 처리를 마음의 구조 탐구에 활용하는 다양한 방법을 소개하는 것을 목표로 합니다.  
누구나 접근하여 공부할 수 있도록 돕는 자료를 만들고자 합니다.  
자세한 사항은 아래 프로젝트 링크를 참고해 주세요. 


## 프로젝트 링크

- https://devwonkwang.github.io/mind-structure-nlp/

## 소개

이 프로젝트에서는 다음과 같은 내용을 다룹니다.

- 네트워크 과학을 활용한 언어 자료 분석
- 토픽 모델링을 활용한 언어 자료 분석
- 신경망과 딥러닝 알고리즘을 활용한 언어 자료 분석
